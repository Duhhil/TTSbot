## English

### Overview

A Discord bot that converts text into speech and plays it in a voice channel. It is built in Python and uses the Google Translate voice service through gTTS.

### How it works

Send a message in chat starting with `}` and the bot will speak it in the voice channel you are currently in. Each user can have their own voice, speed, and volume settings, which are saved automatically.

If several people use the bot at the same time, the requests are queued and played one after another without interrupting anyone. In that case, the bot announces the speaker's name before the message to make it easier to identify who is talking.

### Requirements

- Python 3.8 or higher
- A Discord bot token

### Installation

1. Open the project folder.
2. Install the dependencies:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file in the project root with the following content:

```env
DISCORD_TOKEN=your_token_here
```

4. Start the bot with:

```bash
iniciar_bot.bat
```

> You do not need to install ffmpeg separately, as that is handled by one of the dependencies.

### How to use

In Discord, send a message starting with `}` to make the bot speak. Example:

```text
}Hello, world!
```

### Discord commands

By default, the commands are in English. You can switch to Portuguese with:

```text
}language pt
```

or:

```text
}idioma pt
```

#### English commands (default)

- `}<text>` — speaks the text in the voice channel
- `}voice <language> [tld]` — changes the language and accent, for example: `}voice en com`
- `}speed <value>` — changes the speech speed, from 0.5 to 2.0
- `}volume <value>` — changes the volume, from 0.1 to 2.0
- `}name on` / `}name off` — turns your name announcement on or off when someone else is using the bot
- `}config` — shows your current configuration
- `}resetconfig` — restores the default values
- `}language en` / `}language pt` — changes the bot command language globally
- `}help` — shows the list of commands in Discord

#### Portuguese commands

After using `}language pt`, the commands become:

- `}<texto>`
- `}voz <idioma> [tld]`
- `}velocidade <valor>`
- `}volume <valor>`
- `}nome on/off`
- `}config`
- `}resetconfig`
- `}idioma en/pt`
- `}ajuda`

There is also a 4.5-second cooldown per user between one message and the next to prevent spam.

### Terminal commands

While the bot is running, you can use the following commands directly in the terminal:

- `ajuda` — lists the available commands
- `cls` — clears the screen
- `status` — shows uptime and how many voice channels are connected
- `fila` — shows how many messages are waiting in each server
- `limpar` — removes temporary audio files left in the folder
- `desconectar` — disconnects the bot from all voice channels
- `logs` — shows the last 10 lines of the log
- `sair` — shuts down the bot

### Auto-restart

The `iniciar_bot.bat` file restarts the bot automatically if it stops unexpectedly. To make it start on its own after a system restart or power outage, you can configure Windows to run this file at login, for example with Task Scheduler.

### File structure

- `bot.py` — main bot file
- `requirements.txt` — project dependencies
- `.env` — bot token, which should be kept private
- `configs_usuarios.json` — user settings, created automatically
- `bot_language.json` — current command language, created automatically
- `audios/` — temporary audio files generated by the bot
- `logs/` — log files with automatic rotation

## Português

### Visão geral

Um bot de Discord que converte texto em fala e reproduz a mensagem em um canal de voz. O projeto foi desenvolvido em Python e usa o serviço de voz do Google Translate por meio do gTTS.

### Como funciona

Envie uma mensagem no chat começando com `}` e o bot irá falar essa mensagem no canal de voz em que você estiver. Cada usuário pode ter sua própria configuração de voz, velocidade e volume, salva automaticamente.

Se várias pessoas usarem o bot ao mesmo tempo, as solicitações entram em uma fila e são reproduzidas uma após a outra sem interromper ninguém. Nesse caso, o bot anuncia o nome da pessoa antes da mensagem para facilitar a identificação de quem está falando.

### Requisitos

- Python 3.8 ou superior
- Um token de bot do Discord

### Instalação

1. Abra a pasta do projeto.
2. Instale as dependências:

```bash
pip install -r requirements.txt
```

3. Crie um arquivo `.env` na raiz do projeto com o seguinte conteúdo:

```env
DISCORD_TOKEN=seu_token_aqui
```

4. Inicie o bot com:

```bash
iniciar_bot.bat
```

> Você não precisa instalar o ffmpeg separadamente, pois isso já é tratado por uma das dependências.

### Como usar

No Discord, envie uma mensagem começando com `}` para fazer o bot falar. Exemplo:

```text
}Olá, mundo!
```

### Comandos no Discord

Por padrão, os comandos aparecem em inglês. Você pode trocar para português com:

```text
}language pt
```

ou:

```text
}idioma pt
```

#### Comandos em inglês (padrão)

- `}<text>` — fala o texto no canal de voz
- `}voice <language> [tld]` — altera o idioma e o sotaque, por exemplo: `}voice en com`
- `}speed <value>` — altera a velocidade da fala, de 0.5 a 2.0
- `}volume <value>` — altera o volume, de 0.1 a 2.0
- `}name on` / `}name off` — ativa ou desativa o anúncio do seu nome quando outra pessoa estiver usando o bot
- `}config` — mostra sua configuração atual
- `}resetconfig` — restaura os valores padrão
- `}language en` / `}language pt` — troca o idioma dos comandos do bot de forma global
- `}help` — mostra a lista de comandos no Discord

#### Comandos em português

Depois de usar `}language pt`, os comandos passam a ser:

- `}<texto>`
- `}voz <idioma> [tld]`
- `}velocidade <valor>`
- `}volume <valor>`
- `}nome on/off`
- `}config`
- `}resetconfig`
- `}idioma en/pt`
- `}ajuda`

Também existe um cooldown de 3,5 segundos por usuário entre uma mensagem e a próxima para evitar spam.

### Comandos no terminal

Enquanto o bot estiver rodando, você pode usar os seguintes comandos diretamente no terminal:

- `ajuda` — lista os comandos disponíveis
- `cls` — limpa a tela
- `status` — mostra o tempo de atividade e quantos canais de voz estão conectados
- `fila` — mostra quantas mensagens estão aguardando em cada servidor
- `limpar` — remove arquivos temporários de áudio que ficaram na pasta
- `desconectar` — desconecta o bot de todos os canais de voz
- `logs` — mostra as últimas 10 linhas do log
- `sair` — encerra o bot

### Reinicialização automática

O arquivo `iniciar_bot.bat` reinicia o bot automaticamente caso ele pare inesperadamente. Para que ele suba sozinho após reinicialização do sistema ou queda de energia, você pode configurar o Windows para abrir esse arquivo no login, por exemplo com o Agendador de Tarefas.

### Estrutura de arquivos

- `bot.py` — arquivo principal do bot
- `requirements.txt` — dependências do projeto
- `.env` — token do bot, que deve ser mantido em segredo
- `configs_usuarios.json` — configurações de cada usuário, criadas automaticamente
- `bot_language.json` — idioma atual dos comandos, criado automaticamente
- `audios/` — arquivos temporários de áudio gerados pelo bot
- `logs/` — arquivos de log com rotação automática
